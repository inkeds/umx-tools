# 文档模式

## single-file

输出 1 份文件：`00-single-file-pack.md`。

包含：
- 组合决策
- Epic
- Feature/Story
- Core Spec
- 里程碑
- AI Prompt 包

## minimal

输出多文件最小闭环：

- `00-epic-map.md`
- `01-feature-story-map.md`
- `02-core-spec.md`
- `03-combo-decision.md`
- `04-milestone-plan.md`
- `05-ai-prompt-pack.md`
- `30-*` 组合扩展文档

## standard/full

在 minimal 基础上增加传统工程与治理层文档。
